**Variable types**
-----------------
In case of sending and receiving data, the following variable types are used to clearly show the data length, even if specified enumeration type is present.
```cpp
typedef int32_t   s32;
typedef int16_t   s16;
typedef int8_t    s8;
typedef uint32_t  u32;
typedef uint16_t  u16;
typedef uint8_t   u8;
```
